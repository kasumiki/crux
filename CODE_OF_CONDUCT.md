# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal
attacks are not tolerated. We welcome contributions and questions from everyone.

To report unacceptable behavior, contact the maintainers via a private channel
(see [SECURITY.md](SECURITY.md) for private reporting). Maintainers are responsible for
clarifying standards and may remove contributions or contributors that violate this code.
